Crawler OpenGraph

Hello! This is a HTML crawler for retrieving opengraphs.

OpenGraph is a pattern to describe your website. Google, Facebook, Telegram and too much others applications crawl your website for these information. To turn your websites into graph objects, see how here!

How to use!

To use this crawler is really simple. There are two options.

  • GetOpenGraph

    https://crawl-og.herokuapp.com/api/HtmlCrawl/opengraph?url=[URLHERE]&imgarr=[boolean]

    This service gets a json object with data from the url you provided.

    Params

    • url: Url you want to crawl. No "hashtags" (#) on the url. Only urls that return html.
    • imgarr: "false" for get a single image from the url, "true" for getting all images in html

  • GetImages

    https://crawl-og.herokuapp.com/api/HtmlCrawl/images?url=[URLHERE]

    This service gets an array of images from the provided url

    Params

    • url: Url you want to crawl. No "hashtags" (#) on the url. Only urls that return html.
